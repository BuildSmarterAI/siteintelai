# Security Policy

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | ✅ Active support  |
| < 1.0   | ❌ No support      |

## Reporting a Vulnerability

We take security seriously at SiteIntel™. If you discover a security vulnerability, please report it responsibly.

1.  **Private Disclosure:** Do not publicly disclose the vulnerability.
2.  **Contact:** Email us at [security@siteintel.com](mailto:security@siteintel.com) with detailed information about the vulnerability, including steps to reproduce it.
3.  **Response:** Our security team will acknowledge your report within 48 hours and provide an estimated timeline for addressing the issue.
4.  **Resolution:** We will keep you informed about the progress and notify you when the vulnerability has been resolved.

## Security Measures

### Data Protection
-   **Encryption:** All sensitive data is encrypted both in transit (using HTTPS) and at rest (using AES-256 encryption).
-   **Access Control:** Access to customer data is strictly controlled and limited to authorized personnel only.
-   **Regular Backups:** We perform regular backups of all data to ensure business continuity and prevent data loss.

### Infrastructure Security
-   **Firewalls:** Our infrastructure is protected by firewalls and intrusion detection systems.
-   **Security Audits:** We conduct regular security audits and penetration testing to identify and address potential vulnerabilities.
-   **Monitoring:** We continuously monitor our systems for suspicious activity and security incidents.

### Application Security
-   **Secure Coding Practices:** We follow secure coding practices to prevent common web application vulnerabilities such as SQL injection, cross-site scripting (XSS), and cross-site request forgery (CSRF).
-   **Dependency Management:** We regularly update our dependencies to patch security vulnerabilities.
-   **Authentication and Authorization:** We use strong authentication and authorization mechanisms to protect user accounts and data.

### Compliance
-   **Data Privacy:** We comply with applicable data privacy regulations, including GDPR and CCPA.
-   **Security Standards:** We adhere to industry-standard security frameworks and best practices.

## Responsible Disclosure

We appreciate responsible disclosure of security vulnerabilities. We commit to:

-   Acknowledging receipt of your report within 48 hours.
-   Providing regular updates on the progress of the investigation and resolution.
-   Recognizing your contribution in our security advisories (if you wish).

We ask that you:

-   Provide us with a reasonable timeframe to address the vulnerability before disclosing it publicly.
-   Avoid exploiting the vulnerability or accessing other users' data.
-   Act in good faith and avoid causing harm to our systems or users.

## Security Updates

We regularly release security updates to address identified vulnerabilities. We encourage users to apply these updates promptly to ensure the security of their systems and data.
